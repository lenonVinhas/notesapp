# NotesApp Challenge

## Challenge
Create a public repository with a frontend application, without a database connection, using the attached draft as a reference. It's important that the application is 100% functional and has a pleasant visual appearance (use a design system of your preference). Since it is a draft, feel free to make changes, always prioritizing the best user experience.

## Challenge Objective

To technically test your familiarity with frontend development tools, understanding of business logic, user experience, design sensitivity, code quality and delivery agility.

## About the Application
NotesApp is a tool to help people manage their notes. The functionalities are:
- Create notes with tags
- Search by tags or text
- Archive notes
- Delete notes
- Change language (English and Portuguese)

## Minimum Requirements
- Public git repository (github, gitlab, bitbucket, etc)
- Use React and Typescript (framework of your preference, such as Vite, Next, etc)
- README with instructions on how to run the project
- All application functionalities must work

## Optional Requirements
- Unit tests
- Transition animations
- Code Quality (eslint, for example)
- Dockerfile
